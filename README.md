# Investigating Electrified Transportation Systems

## Introduction
This repository contains scripts and functions adapted or inspired from the book "Electric Powertrain" [1] by Hayes and Goodarzi.

## Vehicle Dynamics
To understand and describe the power and energy requirements of automotive powertrains, we implement physics equations governing vehicle dynamics. We also implement torque and power equations for electric motors found in Electric Vehicles.

Executing the following scripts will generate the mentioned figures from the book.
- simplified-bev-acceleration-profile.py produces Figure 2.7.
- simplified-traction-machine-torque-speed.py produces Figure 2.8.
